# Contributor guidelines
New contributors to this project , please follow these guidelines
- All folder names should be lower cased
- all files and folder names should also be snake cased
- Any new files that get added to the repo must be categorised appropriately. Refer to the project structure if you need help with this or open an issue for further clarification.
